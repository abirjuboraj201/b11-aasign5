Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. They are -

  1. getElementById: It selects an element by it's unique id attribute.
  2. getElementsByClassName: It selects elements by their class name.
  3. getElementsByTagName: It selects elements by their tag names.
  4. getElementsByName: It selects elements by their names.
  5. querySelector: It selects the first element that has the specified css selector and returns null if no such element is found.
  6. querySelectorAll: It selects all the elements that have the specified css selector and returns empty NodeList if no such elements are found.

Question-2: What is the difference between innerHTML, innerText and textContent?

innerHTML allows an user to write basic HTML tags, attributes within any element. innerText helps an user to get or set any text to any element. it does not show if there are any hidden text contents. On the other hand, textContent gives user raw text meaning it will show any hidden text along with visible text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we add a single event listener to the parent element instead of adding event listener to multiple child elements.

Question-4: What is event bubbling in the DOM?

When an event starts at the target element or child element and moves up or bubbles up to the root element or parent element, the phenomena is known as event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

There are several methods to create, add and remove elements using JavaScript. They are -

  1. Create Element
    • document.createElement(tagName)
      Ex.: Const newDiv = document.createElement('div');
  2. Add Element
    • appendChild(): Adds new element to the parent element.
      Ex.: document.body.appendchild(newDiv);
    • append(): Adds multiple elements at once.
      Ex.: document.body.append(newDiv, anotherDiv, 'can also add some texts');
  3. Remove Element
    • removeChild(): Removes a child element from a parent element.
      Ex.: document.body.removeChild(newDiv);
    • remove(): Directly removes the element.
      Ex.: newDiv.remove();
